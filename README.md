### Trosについて

Trosは、多機能で、高互換性、高安定性、高信頼性、高速、軽量、高効率かつ小型であることを目標としている、**Haribote OS(河合秀実)ベースのオペレーティング・システム**です。

本プロジェクトは、**コミュニティの長期的な発展と安定性**を重視しており、すべての開発者が安心して貢献できる環境を目指しています。
